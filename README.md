# Fwetch v2.1
fwetch - Haskell Fetch rewrited in C

> ![Warning]
>
> Correctly works only in *BSD systems

### Dependences:

- clang
- make

### Install:

```bash
git clone https://github.com/SysFugy/fwetch
cd hfetch
make
```
